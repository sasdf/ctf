rev - big maffs

big maffs

Do you have enough memory?

Time

5 hours+
Most of the reversing work is done by my teammate @terrynini.

Behavior

It's a binary that will calculate the flag. But it needs enormous memory and time.

Solution

TL;DR

  1. Reverse the binary
  2. Find a isomorphism between that strange numeral system and Z\mathbb{Z}Z.
  3. Calculate modular Ackermann function.
  4. Reduce the exponent using euler's totient.

Numeral system

The binary do some calculation of big integer with a strange numeral system, When add two numbers, the operation is similar to adding two binary integers. However, the carry is -1 instead of 1.

The operation looks isomorphic to normal Interger Ring Z\mathbb{Z}Z, X=X+1−1X=X+1+1−1−1X=1+X−10=X−X=1−1 \begin{aligned} X &= X + 1 - 1 \\ X &= X + 1 + 1 - 1 - 1 \\ X &= 1 + X - 1 \\ 0 &= X - X = 1 - 1 \\ \end{aligned} XXX0​=X+1−1=X+1+1−1−1=1+X−1=X−X=1−1​

Ackermann function

Now, we can calculate the flag with python's big interger than the strange and slow implementation in that binary. However, it seems it will run forever.

The function is: A(m,n)={n+1if m=0A(m−1,1)if m>0 and n=0A(m−1,A(m,n−1))if m>0 and n=0 A(m, n) = \begin{cases} n + 1 & \text{if } m = 0 \\ A(m - 1, 1) & \text{if } m > 0 \text{ and } n = 0 \\ A(m - 1, A(m, n - 1)) & \text{if } m > 0 \text{ and } n = 0 \\ \end{cases} A(m,n)=⎩⎪⎨⎪⎧​n+1A(m−1,1)A(m−1,A(m,n−1))​if m=0if m>0 and n=0if m>0 and n=0​

Reduce with euler's totient

Euler totient has a property that: xy≡xy+ϕ(N)mod  N (if x, y are coprimes) x^y \equiv x^{y + \phi(N)} \mod N \text{ (if x, y are coprimes)} xy≡xy+ϕ(N)modN (if x, y are coprimes) Moreover, if yyy is larger than ϕ(N)\phi(N)ϕ(N), the congruence holds even if x,yx, yx,y are not coprimes.

Next, Knuth's up-arrow notation is defined as: a↑nb={abif n=11if n≥1 and b=0a↑(n−1)(a↑n(b−1))otherwise a \uparrow^{n} b = \begin{cases} a ^ b & \text{if } n = 1 \\ 1 & \text{if } n \ge 1 \text{ and } b = 0 \\ a \uparrow^{(n-1)} (a \uparrow^{n} (b - 1)) & \text{otherwise}\\ \end{cases} a↑nb=⎩⎪⎨⎪⎧​ab1a↑(n−1)(a↑n(b−1))​if n=1if n≥1 and b=0otherwise​

pp = p
