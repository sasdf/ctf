Solution

The server provide three operations:

  • Encryption(aeskey ^ operand)
  • Encryption(aeskey + operand)
  • AESDec(inputkey, AESEnc(aeskey, flag))

When bit iii in aeskey is 0, aeskey⊕2i=aeskey+2i  ⟹  Encryption(aeskey⊕2i)=Encryption(aeskey+2i)\begin{aligned} & \text{aeskey} \oplus 2^i = \text{aeskey} + 2^i \\ \implies & \text{Encryption}(\text{aeskey} \oplus 2^i) = \text{Encryption}(\text{aeskey} + 2^i) \\ \end{aligned}⟹​aeskey⊕2i=aeskey+2iEncryption(aeskey⊕2i)=Encryption(aeskey+2i)​ Otherwise those are not equal. We can recover the key bit-by-bit to get key for decrypting the flag.
