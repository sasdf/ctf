crypto - Lost Modulus

Lost Modulus

I lost my modulus. Can you find it for me?

Time

2 hours
First blood

Behavior

The server implement paillier cryptosystem. It gives us a encrypted flag at start.

There are two operations, A and B, Operation A return the ciphertext of our input. Operation B gives us the last byte of plaintext. We can run atmost 2048 operations.

Unfortunately, we know nothing about n,p,q,gn, p, q, gn,p,q,g.

Solution

TL;DR

  1. Leak 8 ~ 1024 bits of n bit-by-bit (1008 OPs).
  2. Last byte is -ret (0 OPs).
  3. Leak 16 bytes of flag (16 OPs).
  4. Repeat with a new connection.

We can calculate m mod  nm\ \mod nm modn by encrypting it and then decrypting, If m<nm < nm<n, we get the last byte of $m$. However, if m≥nm \ge nm≥n, we get the last byte of m mod  nm\ \mod nm modn. We can leak nnn bit-by-bit by selecting 2b2^b2b as mmm. Moreover, the return value is −n mod  256-n \ \mod 256−n mod256 if m≥nm \ge nm≥n, which means we have 16 operations left for leaking 16 bytes of flag.
